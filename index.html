Heathrow Hotel Parking — First Time Setup

Complete these 3 steps once. After that every staff member just opens the app and logs in.

Step 1 — Create free Supabase account & project

  1. Go to supabase.com → click Sign up (free)
  2. Click New project → name it heathrow-parking → choose EU West (London) → click Create
  3. Wait about 2 minutes for the green "Project is ready" message

Step 2 — Run the setup SQL

  1. In your Supabase project, click SQL Editor in the left sidebar
  2. Click + New query
  3. Click anywhere inside the box below to select all the SQL, then copy it (Ctrl+C)
  4. Paste it into the SQL Editor and click the green Run button (or press Ctrl+Enter)
  5. You should see "Success. No rows returned."

📧 Automatic Email Notifications

How to set up automatic emails (5 minutes, free):
  1. Go to resend.com → Sign up free (100 emails/day free forever)
  2. In Resend dashboard → API Keys → Create API Key → copy the key (starts with re_)
  3. Go to Domains → Add your domain (e.g. arielhotel.co.uk) and verify the DNS records — or use Resend's free shared domain to start
  4. Paste the key below → enter your From address → click Save & Activate
